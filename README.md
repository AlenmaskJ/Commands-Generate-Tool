# Commands-Generate-Tool
### 痛点：
#### 当需要生成根据不同的号码去修改某一参数的命令时候，客户只给你一个excel表格包括了需要的所有修改号码，让你去批量修改

#### 单文件版：
https://github.com/AlenmaskJ/Commands-Generate-Tool/releases/download/V1.0/CommandGenerator.exe
#### ZIP文件版：
https://github.com/AlenmaskJ/Commands-Generate-Tool/releases/download/V1.0/CommandGenerator.zip

### 作用：
#### 本工具能根据自定义命令模板，进行拼接生成命令，实现批量生成命令的需求
